# Kompakt Audio Snippets

Generate audio snippets from the command line

## Install

+ `git clone https://github.com/kompakt/audio-snippets.git`
+ `cd audio-snippets`
+ `curl -sS https://getcomposer.org/installer | php`
+ `php composer.phar install`

## Test

+ `vendor/bin/phpunit`
